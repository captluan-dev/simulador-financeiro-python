# Simulador Financeiro

Projeto desenvolvido em Python com o objetivo de simular decisões financeiras como investimentos, dívidas e inflação.

## Funcionalidades

- Aplicação de juros em investimentos
- Aplicação de juros em dívidas
- Simulação de inflação
- Sistema de eventos financeiros

## Tecnologias utilizadas

- Python 3.13.11

## Estrutura do Projeto

main.py → Arquivo principal
jogador.py → Controle do jogador
sistema_financeiro.py → Regras financeiras
eventos.py → Eventos aleatórios
config.py → Lista com os meses
utils.py → Utilidades extras

## Como executar o projeto

1. Clone o repositório:

git clone https://github.com/captluan-dev/simulador-financeiro-python.git

2. Entre na pasta:

cd simulador-financeiro-python

3. Execute:

python main.py

## Objetivo do Projeto

Projeto criado para praticar lógica de programação, organização de código e conceitos financeiros básicos.
